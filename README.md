## Running For Development

```
$ npm install
$ npm run dev
```

## Building

```
$ npm run build
```

## Deploying

Copy `aws-credentials.json.template` as `aws-credentials.json`  
Fill missing values in `aws-credentials.json`  
```
$ npm run deploy
```
